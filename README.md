# fruitstandsoftware.github.io

Official website for [fruitstandsoftware.com](https://fruitstandsoftware.com), hosted with GitHub Pages.

## Local preview

From the repository root, run:

```bash
python3 scripts/build_localized_site.py
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

The site publishes locale-specific pages under full locale codes such as:

- `/en-US/`
- `/en-GB/`
- `/es-ES/`
- `/ja/`

The root `index.html` is a locale redirect shell generated by `scripts/build_localized_site.py`.

## Deployment

This repo is published directly through GitHub Pages. Pushes to the configured Pages branch deploy automatically.
