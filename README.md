# fruitstandsoftware.github.io

Official website for [fruitstandsoftware.com](https://fruitstandsoftware.com), hosted with GitHub Pages.

## Local preview

From the repository root, run:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Deployment

This repo is published directly through GitHub Pages. Pushes to the configured Pages branch deploy automatically.
